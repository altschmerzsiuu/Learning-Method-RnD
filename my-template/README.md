# 🎓 Education Template

Template HTML modern untuk website edukasi, terinspirasi dari desain website [Breakfast](https://wearebreakfast.com/).

## 🚀 Quick Start

1. Buka file `index.html` di browser
2. Edit konten sesuai kebutuhanmu
3. Ganti placeholder dengan gambar/logo aslimu

## 📁 Struktur Folder

```
my-template/
├── index.html          # Halaman utama
├── css/
│   ├── variables.css   # Warna, font, spacing
│   ├── base.css        # Reset & base styles
│   ├── typography.css  # Font styles
│   ├── layout.css      # Grid & flex utilities
│   ├── animations.css  # Animasi CSS
│   ├── components.css  # Komponen UI
│   └── responsive.css  # Media queries
├── js/
│   └── main.js         # JavaScript interaktif
└── assets/
    ├── images/         # Taruh gambar di sini
    └── icons/          # Taruh icon di sini
```

## 🎨 Cara Custom

### Ganti Logo
Cari komentar `<!-- PUT YOUR LOGO HERE -->` di index.html

### Ganti Warna
Edit file `css/variables.css`:
```css
:root {
  --color-salmon: #e89d9f;    /* Ganti warna di sini */
  --color-yellow: #ffde30;
  /* ... */
}
```

### Ganti Font
Edit file `css/typography.css` line pertama:
```css
@import url('https://fonts.googleapis.com/css2?family=YourFont...');
```

### Tambah Section Baru
Copy struktur section yang sudah ada di `index.html`
